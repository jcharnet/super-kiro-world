# Product Overview

Super Kiro World is a browser-based platformer game themed around Kiro debugging cloud systems at AWS re:Invent. The game features:

- **Core Mechanic**: Time Warp - players can rewind 2-3 seconds of gameplay with limited uses per level
- **Objective**: Navigate platforms representing cloud services (dashboards, pipelines, services), collect metrics, and reach the Deploy Gate
- **Theme**: AWS re:Invent expo environment with cloud infrastructure metaphors
- **Target Audience**: Workshop participants at AWS re:Invent learning to build games

## Key Features

- Smooth platforming physics with collision detection
- Time Warp mechanic with visual feedback (particles, ghost trails)
- Score persistence and leaderboard system
- Parallax scrolling backgrounds
- Screen shake and particle effects
- Lives system with respawn mechanics
- Collectible metrics that contribute to score
- Deploy Gate as level completion objective

## User Experience

- Retro pixel art aesthetic with modern polish
- Kiro brand colors (purple #790ECB) for UI and accents
- Dark theme with high contrast for accessibility
- Immediate response to player input for satisfying game feel
